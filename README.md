🌍 GeoScope (WorldAtlas)

GeoScope is a modern React-based frontend application that allows users to explore country data including population, capital, region, and more through an interactive and responsive UI.

⸻

🚀 Features

* 🌐 Browse all countries with key information
* 🔍 Search countries by name (with debounce optimization)
* 🌎 Filter countries by region
* 🔃 Sort countries (A–Z / Z–A)
* 📄 View detailed information for each country
* 📱 Fully responsive design (mobile-first)
* ⚡ Optimized rendering using React hooks

⸻

🛠 Tech Stack

* React.js (Vite)
* JavaScript (ES6+)
* CSS (Responsive + Modern Layouts)
* Axios (API handling)
* React Router (Navigation)
* REST Countries API
